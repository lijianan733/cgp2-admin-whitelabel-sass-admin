# cgp2-admin-whitelabel-sass-admin

本仓库为项目根（从本地工作区直接提交到 GitHub）。

快速推送到远程仓库（在项目根运行以下命令）：

```bash
# 在项目根目录执行：
git init
git add -A
git commit -m "Initial commit"
git remote add origin https://github.com/lijianan733/cgp2-admin-whitelabel-sass-admin.git
git branch -M master
git push -u origin master
```

Windows PowerShell 等价命令（在 PowerShell 或 Git Bash 中运行）：

```powershell
git init
```

注意：如果远程仓库已有提交或需要身份验证，请根据 Git 的提示使用合适的凭据（例如 GitHub Personal Access Token）。
